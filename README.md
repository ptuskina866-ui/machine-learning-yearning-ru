# Эндрю Нг — Машинное обучение в деталях (Machine Learning Yearning) 🚀

Добро пожаловать в репозиторий с полным профессиональным переводом книги **Эндрю Нг «Machine Learning Yearning»** на русский язык.

Эта книга — незаменимое руководство по технической стратегии для ИИ-инженеров в эпоху глубокого обучения. Она помогает понять, как правильно приоритизировать задачи в проектах машинного обучения, настраивать выборки и анализировать ошибки.

## 🗺️ Краткое содержание (Главы 1–58)

1. **Часть I. Настройка выборок для разработки (dev) и тестирования (test)** — как избежать несовпадения распределений данных.
2. **Часть II. Базовый анализ ошибок** — как ручной анализ 100 ошибок экономит месяцы разработки.
3. **Часть III. Смещение и дисперсия (Bias & Variance)** — стратегии борьбы с недообучением и переобучением.
4. **Часть IV. Кривые обучения** — визуальная диагностика ИИ-систем.
5. **Часть V. Сквозное глубокое обучение (End-to-end deep learning)** — плюсы, минусы и редизайн конвейеров (пайплайнов).
